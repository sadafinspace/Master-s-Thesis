Each file represents the GCS fitted data for the event on date as indicated by the file name (yyyy-mm-dd).
These .rt files should open with any editor (e.g. gedit)

Parameter details:
DATE -- date of observation
TIME -- time of observation 
LON  -- carrington longitude from GCS fit
LAT  -- Laitude from GCS fit   
ROT  -- Tilt angle from GCS fit (in deg)
HEIGHT-- Heliocentric distance of the leading edge in Rs (solar radii units) from GCS
RATIO --  Aspect ratio from GCS fit
HALF ANGLE-- Half angle from GCS fit (in deg) 

The last column indicates the observing coronagraph
C2- STEREO COR2
HI1/2 - STEREO HI1 and HI2
